# 安全模式新UI - 键盘快捷键

## 全局快捷键

### 导航

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+1` | 聚焦左侧面板 | 跳转到文件树面板 |
| `Ctrl+2` | 聚焦中间面板 | 跳转到会话列表面板 |
| `Ctrl+3` | 聚焦右侧面板 | 跳转到终端面板 |
| `Ctrl+F` | 聚焦搜索框 | 快速搜索文件 |

### 操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+M` | 合并到项目 | 触发当前激活会话的合并操作 |
| `Ctrl+Shift+Z` | 回滚已合并 | 触发当前激活会话的回滚操作 |
| `Esc` | 关闭 Diff 面板 | 关闭当前打开的文件 diff 视图 |

## 面板内快捷键

### 文件树面板（左侧）

| 操作 | 说明 |
|------|------|
| 点击文件 | 在中间面板打开 diff 视图 |
| 点击文件夹 | 展开/折叠文件夹 |
| 搜索框输入 | 实时过滤文件列表 |

### 会话列表面板（中间）

| 操作 | 说明 |
|------|------|
| 点击会话标题 | 激活该会话 |
| 点击展开按钮 | 展开/折叠会话详情 |
| 点击文件项 | 打开该文件的 diff 视图 |
| 点击"合并到项目" | 合并当前会话的所有待合并文件 |
| 点击"回滚已合并" | 回滚当前会话已合并的文件（需确认） |

### Diff 面板（中间，覆盖会话列表）

| 操作 | 说明 |
|------|------|
| 点击 hunk checkbox | 选择/取消选择该 hunk |
| 点击"全选" | 选择当前文件的所有 hunks |
| 点击"取消全选" | 取消选择当前文件的所有 hunks |
| 点击关闭按钮 | 关闭 diff 面板 |
| `Esc` | 关闭 diff 面板 |

### 终端面板（右侧）

| 操作 | 说明 |
|------|------|
| 滚动 | 查看历史命令输出 |
| 点击命令项 | 展开/折叠命令详情 |

## 鼠标操作

### 面板调整

| 操作 | 说明 |
|------|------|
| 拖拽左侧分隔条 | 调整文件树面板宽度（200-600px） |
| 拖拽右侧分隔条 | 调整终端面板宽度（200-600px） |

### 文件操作

| 操作 | 说明 |
|------|------|
| 单击文件 | 打开 diff 视图 |
| 悬停文件 | 高亮显示 |

### Hunk 操作

| 操作 | 说明 |
|------|------|
| 单击 checkbox | 选择/取消选择 hunk |
| 单击"全选" | 选择所有 hunks |
| 单击"取消全选" | 取消所有选择 |

## 状态指示

### 文件状态图标

| 图标 | 状态 | 颜色 |
|------|------|------|
| ➕ | 新增文件 | 绿色 |
| ✏️ | 修改文件 | 黄色 |
| ❌ | 删除文件 | 红色 |
| ⚠️ | 冲突文件 | 橙色 |
| ✓ | 已合并文件 | 绿色 |

### 按钮状态

| 状态 | 说明 |
|------|------|
| 正常 | 可点击 |
| 禁用（灰色） | 不可点击（无待合并文件或正在执行操作） |
| 加载中 | 显示旋转图标和"合并中..."/"回滚中..." |

### Toast 通知

| 类型 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 成功 | ✓ | 绿色 | 操作成功完成 |
| 错误 | ✕ | 红色 | 操作失败，显示错误信息 |
| 信息 | ℹ | 蓝色 | 一般信息提示 |

## 过滤模式

### 文件树过滤

| 模式 | 说明 |
|------|------|
| All | 显示所有文件（包括未变更） |
| Changed | 仅显示变更文件（默认） |
| Pending | 仅显示待合并文件 |
| Conflicts | 仅显示冲突文件 |

## Diff 模式

| 模式 | 说明 | 可用性 |
|------|------|--------|
| 原项目 vs 克隆 | 对比原始项目和克隆项目的差异 | ✅ 可用 |
| 克隆历史 | 查看克隆项目的变更历史 | ✅ 可用 |
| 三路合并 | 三路合并视图（冲突解决） | ⏳ Phase 4+ |

## 提示

1. **Hunk 选择持久化**：选中的 hunks 会自动保存到 sessionStorage，刷新页面后仍然保留
2. **面板宽度持久化**：调整的面板宽度会自动保存到 localStorage
3. **会话展开状态持久化**：展开的会话状态会自动保存到 localStorage
4. **合并前确认**：回滚操作会弹出确认对话框，防止误操作
5. **错误提示**：操作失败时会显示详细的错误信息（Toast 通知，5秒后自动消失）
6. **成功提示**：操作成功时会显示简短的成功信息（Toast 通知，3秒后自动消失）

## 常见工作流

### 1. 查看和合并单个文件

1. 在左侧文件树或中间会话列表中点击文件
2. 在 Diff 面板中查看变更
3. （可选）选择特定的 hunks
4. 点击"合并到项目"按钮
5. 等待操作完成，查看成功提示

### 2. 批量合并所有文件

1. 在中间会话列表中展开目标会话
2. 直接点击"合并到项目"按钮（不选择 hunks）
3. 等待操作完成，查看成功提示

### 3. 选择性合并（Hunk 级别）

1. 在左侧文件树或中间会话列表中点击文件
2. 在 Diff 面板中勾选需要合并的 hunks
3. 关闭 Diff 面板（或打开其他文件继续选择）
4. 点击"合并到项目"按钮
5. 仅选中的 hunks 会被合并

### 4. 回滚已合并的文件

1. 在中间会话列表中展开目标会话
2. 点击"回滚已合并"按钮
3. 在确认对话框中点击"确定"
4. 等待操作完成，查看成功提示

### 5. 搜索文件

1. 按 `Ctrl+F` 或点击左侧搜索框
2. 输入文件名关键字
3. 文件列表实时过滤
4. 点击文件查看 diff

### 6. 过滤文件

1. 在左侧文件树顶部选择过滤模式
2. 文件列表根据选择的模式过滤
3. 点击文件查看 diff
